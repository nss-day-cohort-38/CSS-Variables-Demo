Basic Usage

style.css - declare the custom property

:root {

--primary-bg-color: rgb(184, 23, 144);

}

style.css - apply the custom property to whichever elements you want

nav {

background-color: var(--primary-bg-color);

}

Importing Other Stylesheets

buttons.css - additional style sheet (notice the use of the global variable styles)

:root {

--btn-font-family: monospace;

}

button {

font-family: var(--btn-font-family);

background-color: var(--accent-color);

color: var(--primary-color-text);

}

style.css - import button.css into style.css

@import "./buttons.css";
